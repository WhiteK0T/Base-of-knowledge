# 🎬 ytDownloader

[**ytDownloader**](https://github.com/aandrew-me/ytDownloader) (автор — aandrew-me, лицензия **GPL-3.0**) — кроссплатформенный GUI поверх **yt-dlp** + **FFmpeg**. Стек **Electron**, ~9.7k★, активно развивается. Без трекеров и рекламы.

## ✨ Возможности

- Сотни сайтов (YouTube, Facebook, Instagram, TikTok, Twitter/X и др.).
- **Сжатие видео с аппаратным ускорением**.
- Выбор диапазона (обрезка по времени), субтитры, плейлисты.
- Несколько тем оформления.

## 🖥️ Платформы и установка

- **Linux:** AppImage, **Flatpak**, Snap.
- **Windows:** EXE/MSI, Chocolatey, Scoop, **winget**.
- **macOS:** DMG (Intel/ARM64).

**Когда брать:** хочется GUI с дополнительными плюшками поверх скачивания — встроенное сжатие/обрезка, удобная установка через пакетные менеджеры. Сравнение — [Загрузчики — какую выбрать](Загрузчики%20%E2%80%94%20какую%20выбрать.md).
