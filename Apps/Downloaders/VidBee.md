# 🐝 VidBee — загрузчик видео (GUI поверх yt-dlp)

[**VidBee**](https://github.com/nexmoe/VidBee) (автор — nexmoe, лицензия **MIT**) — современный открытый **загрузчик видео** с 1000+ поддерживаемых сайтов. По сути — удобный графический фронтенд поверх **yt-dlp** и **FFmpeg**, который избавляет от возни с командной строкой. ~9.3k★, активно развивается.

## ✨ Возможности

- Скачивание с YouTube, TikTok, Instagram, Twitter/X и **1000+** других платформ (через yt-dlp).
- **RSS-автозагрузка** — фоновые подписки, новые видео качаются автоматически.
- Выбор формата на выходе: **MP4 / MKV / WebM / Auto**.
- Очередь загрузок с прогрессом в реальном времени; пауза/возобновление/повтор в один клик.

## 🖥️ Платформы и запуск

- **Desktop** — Electron (Win/macOS/Linux). Скачать сборки: [vidbee.org](https://vidbee.org).
- **Web** — TanStack Start; локально: `pnpm run start:web`.
- **Browser extension** — на WXT.
- **Docker** — готовый API через `docker-compose`.

**Стек:** TypeScript, Electron, React, Vite, Tailwind CSS, shadcn/ui.

> [!note] Зависимости
> Под капотом — **yt-dlp** (логика извлечения) и **FFmpeg** (склейка/перекодирование). Качество и список рабочих сайтов фактически определяются версией yt-dlp.

> [!warning] Правовой момент
> Скачивай только тот контент, на который у тебя есть право, и учитывай условия использования площадок и авторские права. Инструмент нейтрален — ответственность за использование на пользователе.
