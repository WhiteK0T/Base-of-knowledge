# 🧭 Загрузчики видео — какой выбрать (2026)

> [!info] Главное
> Движок один — **[yt-dlp](yt-dlp.md)** (CLI). Почти все GUI ниже — лишь удобные оболочки поверх него + FFmpeg. Веб-сервисы (типа cobalt) с середины 2025 часто **блокируются YouTube**, а yt-dlp чинится патчами — поэтому для надёжности берут yt-dlp-инструменты.

## 📊 Сравнение

| Инструмент | Лицензия | Платформы | Стек | Чем хорош |
| :--- | :--- | :--- | :--- | :--- |
| [yt-dlp](yt-dlp.md) | Unlicense | кроссплатформенный (CLI) | Python | Движок-эталон, максимум контроля |
| [VidBee](VidBee.md) | MIT | Win/macOS/Linux + Web + Docker | Electron | 1000+ сайтов, **RSS-автозагрузка** |
| [Open Video Downloader](Open%20Video%20Downloader.md) | AGPL-3.0 | Win/macOS/Linux | Tauri/Rust (форк — Electron) | Простой GUI, пакетная загрузка, до 32 потоков |
| [ytDownloader](ytDownloader.md) | GPL-3.0 | Win/macOS/Linux | Electron | **Сжатие с HW-ускорением**, обрезка, удобная установка |
| [Tartube](Tartube.md) | LGPL-2.1 | Win/Linux/BSD/macOS | Python/GTK | **Подписки на каналы**, мониторинг стримов, библиотека |
| [Seal](Seal.md) | GPL-3.0 | **Android** | Kotlin/Compose | Лучшая открытая качалка для телефона |

## 💎 Рекомендации

| Если нужно… | Бери |
| :--- | :--- |
| Максимум контроля / скрипты / надёжность | **yt-dlp** (CLI) |
| Простой GUI «вставил ссылку → скачал» | Open Video Downloader, VidBee |
| Сжатие/обрезка прямо при скачивании | ytDownloader |
| Следить за каналами как за подписками | Tartube |
| Скачивать на Android | Seal |
| Авто-подписки по RSS на десктопе | VidBee |

> [!warning] Правовой момент
> Скачивай только тот контент, на который имеешь право; учитывай условия использования площадок и авторские права.

> [!note] Прочее (вне списка)
> **Stacher** — полированная оболочка над yt-dlp, но **закрытая** (freeware). **Cobalt** — open source веб/self-host, privacy-first, но **YouTube часто блокирует**. **JDownloader** — мощный, но **не open source** и с adware-инсталлятором.
