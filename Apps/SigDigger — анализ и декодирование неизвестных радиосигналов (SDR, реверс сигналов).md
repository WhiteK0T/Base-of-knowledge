# 📡 SigDigger — анализ и декодирование неизвестных радиосигналов (SDR, реверс сигналов)

[**SigDigger**](https://github.com/BatchDrake/SigDigger) (BatchDrake, **Qt/C++**, лицензия **LGPL-3.0**, ~2469★) — открытый **цифровой анализатор радиосигналов с графическим интерфейсом**. В отличие от обычного SDR-приёмника (который просто даёт «послушать»), SigDigger заточен под **исследование и декодирование незнакомых сигналов**: показывает спектр/водопад, в реальном времени демодулирует **FSK / PSK / ASK**, разбирает «пакетные» (bursty) сигналы, декодирует аналоговое видео и слушает аналоговые голосовые каналы. По сути — инструмент для **реверс-инжиниринга радиоэфира**. Работает с SDR-железом через **SoapySDR** (RTL-SDR, HackRF и др.). Построен на собственных DSP-библиотеках автора — **Sigutils / Suscan / SuWidgets**.

> [!note] Чем отличается от Gqrx / SDR#
> **Gqrx, SDR#** — это «радиоприёмники»: настроился на частоту, послушал/посмотрел спектр. **SigDigger** идёт дальше — **вытаскивает данные из сигнала**: демодулирует цифру (FSK/PSK/ASK), показывает символы, помогает понять модуляцию/скорость/кодировку незнакомого сигнала. Это ближе к задаче «что это за сигнал и как его раскодировать», а не «послушать радио».

## ✨ Что умеет

- **Спектр и водопад** (визуализация, унаследована от Gqrx) — увидеть, где и какой сигнал.
- **Реал-тайм демодуляция** FSK / PSK / ASK с настраиваемыми параметрами.
- **Инспекторы** сигналов: оценка символьной скорости, созвездие, гистограммы — чтобы разобрать незнакомую модуляцию.
- **Декодирование аналогового видео**, прослушивание аналогового голоса.
- **Анализ пакетных/burst-сигналов** (короткие посылки телеметрии, пультов и т.п.).
- **Работа с записями (IQ-файлы)** — можно анализировать сохранённый эфир **без подключённого железа**.

> [!info] Нужно SDR-железо (или запись)
> Сама программа сигналы из воздуха не берёт — нужен **SDR-приёмник** (дешёвый **RTL-SDR** ~30$ на приём; **HackRF/LimeSDR** — приём+передача) и антенна, подключаемые через **SoapySDR**. Без железа SigDigger полезен только для разбора уже **записанных IQ-файлов**.

## ⚖️ Правовой нюанс

> [!caution] Слушать можно не всё
> Приём «вообще эфира» технически не ограничен, но **декодирование чужих/закрытых переговоров** (сотовая связь, служебные/шифрованные каналы, пейджинг с личными данными) подпадает под **тайну связи — ст. 138 УК РФ**. Безопасно и легально: **свои** передатчики, любительские диапазоны (ham), нелицензируемый **ISM** (433/868 МГц — свои датчики/пульты), открытое вещание (FM, метеоспутники NOAA/Meteor, ADS-B самолётов, AIS судов). Реверсь **свои** устройства, а не чужие коммуникации.

## 📦 Установка

Официальной поддержки Windows нет (автор отмечает проблемы с воспроизводимой сборкой DLL). Цель — **GNU/Linux и macOS**. Самый простой путь — **AppImage**; «правильный» распространённый способ собрать весь набор (sigutils→suscan→suwidgets→SigDigger) — скрипт **[`blsd`](https://actinid.org/blsd)**.

| Система | Как поставить |
| :--- | :--- |
| **Gentoo** (основная) | В officiale Portage пакета нет → **AppImage** из релизов **или** сборка скриптом `blsd`. Драйверы железа из Portage: `emerge net-wireless/soapysdr net-wireless/rtl-sdr` (+ `net-wireless/hackrf` при наличии HackRF) |
| **Debian / Ubuntu** | **AppImage** (без зависимостей) или сборка: `sudo apt install build-essential cmake qtbase5-dev libsoapysdr-dev libfftw3-dev libxml2-dev libsndfile1-dev libvolk2-dev librtlsdr-dev portaudio19-dev`, затем `blsd` |
| **Arch** (план с июня 2026) | В **официальном репо**: `sudo pacman -S sigdigger` (стабильная 0.3.0). Свежак — AUR `sigdigger-git` (сначала собрать `sigutils-git`, `suscan-git`, `suwidgets-git`) |
| **Entware / RT-AX56U** | **N/A** — Qt-GUI + нужно SDR-железо и рабочий стол; на роутере (armv7, 512 МБ) не запустить. Роутер максимум может отдавать поток с USB-SDR по сети (rtl_tcp/SoapyRemote), а анализ — на десктопе |

> [!tip] Стабильная версия старая — бери rolling
> Тег стабильного релиза `0.3.0` — ещё 2022 года; актуальная «latest»-сборка катится отдельно (последняя ~апрель 2025). Для свежих фич и фиксов собирай из `master` через `blsd` или ставь AppImage из последнего релиза.

## 💡 Кому полезно

- **Радиолюбителям / SDR-энтузиастам:** разобрать незнакомый сигнал — модуляция, скорость, что внутри.
- **Хобби-реверс железа:** понять протокол **своего** ISM-пульта/датчика (433/868 МГц).
- **Образование/DSP:** наглядно увидеть созвездия, символы, демодуляцию в реальном времени.
- **Не** «волшебная кнопка» (нужны железо/антенна и понимание основ DSP) и **не** для прослушки чужих переговоров.
