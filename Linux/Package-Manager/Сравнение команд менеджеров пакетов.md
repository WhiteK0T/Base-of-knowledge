#### Сравнение команд менеджеров пакетов

| Действие                     | apt (Debian, Ubuntu)                   | dnf (Fedora, RHEL)         | pacman (Arch Linux)          | zypper (openSUSE)                    | opkg (OpenWrt)                     | apk (Alpine)           | portage (Gentoo)                                |
|------------------------------|----------------------------------------|----------------------------|------------------------------|--------------------------------------|------------------------------------|------------------------|-------------------------------------------------|
| Обновить список пакетов      | `apt update`                           | `dnf makecache`            | `pacman -Sy`                 | `zypper refresh`                     | `opkg update`                      | `apk update`           | `emerge --sync`                                 |
| Обновить все пакеты          | `apt upgrade`                          | `dnf upgrade`              | `pacman -Syu`                | `zypper update`                      | `opkg upgrade`                     | `apk upgrade`          | `emerge -uDN @world`                            |
| Установить пакет             | `apt install <пакет>`                  | `dnf install <пакет>`      | `pacman -S <пакет>`          | `zypper install <пакет>`             | `opkg install <пакет>`             | `apk add <пакет>`      | `emerge <пакет>`                                |
| Удалить пакет                | `apt remove <пакет>`                   | `dnf remove <пакет>`       | `pacman -R <пакет>`          | `zypper remove <пакет>`              | `opkg remove <пакет>`              | `apk del <пакет>`      | `emerge --unmerge <пакет>`                      |
| Поиск пакета                 | `apt search <запрос>`                  | `dnf search <запрос>`      | `pacman -Ss <запрос>`        | `zypper search <запрос>`             | `opkg find <запрос>`               | `apk search <запрос>`  | `eix <запрос>` ¹                                |
| Показать информацию о пакете | `apt show <пакет>`                     | `dnf info <пакет>`         | `pacman -Si <пакет>`         | `zypper info <пакет>`                | `opkg info <пакет>`                | `apk info <пакет>`     | `equery meta <пакет>` ¹                         |
| Показать обновляемые пакеты  | `apt list --upgradable`                | `dnf check-update`         | `pacman -Qu`                 | `zypper list-updates`                | `opkg list-upgradable`             | `apk version -l '<'`   | `emerge -puDN @world`                           |
| Удалить осиротевшие пакеты   | `apt autoremove`                       | `dnf autoremove`           | `pacman -Rns $(pacman -Qdt)` | (нет прямой команды) ²               | `opkg remove --autoremove`         | (нет прямой команды) ² | `emerge --depclean`                             |
| Показать осиротевшие пакеты  | `deborphan` ¹                          | `dnf repoquery --unneeded` | `pacman -Qdt`                | `zypper packages --unneeded`         | (нет прямой команды)               | (нет прямой команды)   | `emerge -p --depclean`                          |
| Удалить пакет и осиротевшие  | `apt remove <пакет> && apt autoremove` | `dnf remove <пакет>`       | `pacman -Rns <пакет>`        | `zypper remove --clean-deps <пакет>` | `opkg remove --autoremove <пакет>` | `apk del <пакет>`      | `emerge --unmerge <пакет> && emerge --depclean` |

**Примечания:**
- ¹ Не встроено в пакетный менеджер: `eix` (`app-portage/eix`), `equery` (`app-portage/gentoolkit`), `deborphan` ставятся отдельно. Встроенный поиск в Gentoo — `emerge -s <запрос>` (по имени) / `emerge -S` (по описанию).
- ² Единой встроенной команды нет: в zypper осиротевшие сначала смотрят (`zypper packages --unneeded`), затем удаляют вручную; в Alpine `apk del <пакет>` сам убирает ставшие ненужными зависимости.
- В Gentoo `@world` — набор явно установленных пакетов; `-uDN` = update + deep + newuse.
