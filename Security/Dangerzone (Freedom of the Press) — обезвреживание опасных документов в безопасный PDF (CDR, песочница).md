# 🧯 Dangerzone — обезвреживание опасных документов в безопасный PDF (CDR)

[**Dangerzone**](https://github.com/freedomofpress/dangerzone) ([Freedom of the Press Foundation](https://dangerzone.rocks), **Python**, лицензия **AGPL-3.0**, ~5.6k★, v0.11.0 от 02.07.2026) — открытый инструмент **CDR (Content Disarm & Reconstruction)**: берёт потенциально опасный документ (PDF, офисные файлы, картинки) и превращает его в **гарантированно безопасный PDF**, вырезая любой активный контент. Сделан командой, стоящей за **SecureDrop** — то есть для журналистов и людей с реальной моделью угроз. Прошёл **аудит безопасности** (Include Security, дек. 2023) без high-risk находок.

## 🔬 Как это работает

Идея: **не доверять** входному файлу и «пересобрать» его из **пикселей**, чтобы никакой встроенный код не пережил конвертацию.

1. **Внутри песочницы (untrusted):** исходный документ рендерится в PDF, затем **разбирается в сырые пиксели** (RGB по каждой странице).
2. **Снаружи (trusted):** из пиксельных данных собирается **новый чистый PDF** — в нём только картинка страниц, без макросов, скриптов, эксплойтов, вложений.
3. **Опционально — OCR:** можно вернуть **текстовый слой** (поиск/копипаст), распознав текст с картинок.

> [!info] Песочница — Podman + gVisor, не полноценная ВМ
> Изоляция строится на **контейнерах (Podman)** и **gVisor** (application-kernel на Go, перехватывает системные вызовы Linux) — это лёгкая, но крепкая изоляция без накладных расходов полной виртуалки. Даже если файл эксплуатирует уязвимость конвертера — вредонос остаётся **запертым в одноразовом контейнере**, а наружу выходят только «глупые» пиксели.

## 📄 Что поддерживает

- **Документы:** PDF, Word (`.docx/.doc`), Excel (`.xlsx/.xls`), PowerPoint (`.pptx/.ppt`), ODF (`.odt/.ods/.odp/.odg`), Hancom HWP, **EPUB**.
- **Изображения:** JPEG, GIF, PNG, SVG, BMP, TIFF и др.
- **Платформы:** Windows, macOS, Ubuntu/Debian, Fedora, **Qubes OS** (beta), **Tails**.

## ⚠️ Честные ограничения

> [!caution] Безопасность ≠ «всё сохранится как было»
> Пересборка в пиксели **убивает интерактив**: формы, гиперссылки, встроенные объекты, слои — **пропадают** (остаётся «фотография» страниц + опционально OCR-текст). Также:
> - **Медленно:** песочница + рендеринг добавляют время (большие/многостраничные файлы — ощутимо).
> - **Размер/качество:** итоговый PDF — растровый; текст после OCR может содержать **ошибки распознавания**.
> - **Не 100%-гарантия.** Авторы честно не обещают абсолют: изоляция крепкая, но это защита от **document-borne** эксплойтов/макросов, а не от того, что тебя обманут **самим содержимым** (фишинговая ссылка, нарисованная на «безопасной» странице, всё ещё фишинг).

## 💡 Когда применять

- Открываешь **вложение от незнакомца** (резюме, «счёт», «договор.docx», странный PDF) — прогони через Dangerzone и читай уже безопасную версию.
- Работа с документами из **недоверенных источников** (утечки, архивы, скачанное).
- Часть **opsec-гигиены** наравне с [privacy.sexy](privacy.sexy%20%E2%80%94%20%D0%B3%D0%B5%D0%BD%D0%B5%D1%80%D0%B0%D1%82%D0%BE%D1%80%20%D1%81%D0%BA%D1%80%D0%B8%D0%BF%D1%82%D0%BE%D0%B2%20%D0%BF%D1%80%D0%B8%D0%B2%D0%B0%D1%82%D0%BD%D0%BE%D1%81%D1%82%D0%B8%20%D0%B8%20%D1%85%D0%B0%D1%80%D0%B4%D0%BD%D0%B5%D0%BD%D0%B8%D0%BD%D0%B3%D0%B0%20%D0%B4%D0%BB%D1%8F%20Windows-macOS-Linux%20%28%D1%80%D0%B0%D0%B7%D0%B1%D0%BE%D1%80%2C%20%D0%BD%D1%8E%D0%B0%D0%BD%D1%81%D1%8B%29.md). Если нужно не «обезвредить», а **изучить** malware — это уже [Reversecore MCP](Reverse-Engineering/Reversecore%20MCP%20%E2%80%94%20MCP-%D1%81%D0%B5%D1%80%D0%B2%D0%B5%D1%80%20%D0%B4%D0%BB%D1%8F%20%D1%80%D0%B5%D0%B2%D0%B5%D1%80%D1%81-%D0%B8%D0%BD%D0%B6%D0%B8%D0%BD%D0%B8%D1%80%D0%B8%D0%BD%D0%B3%D0%B0%20%D0%B8%20%D0%B0%D0%BD%D0%B0%D0%BB%D0%B8%D0%B7%D0%B0%20malware%20%28Radare2-r2ghidra-YARA%29.md).

## 🖥️ По системам

| Система | Установка |
| :--- | :--- |
| **Gentoo** (основная) | Нужен **Podman** (`emerge app-containers/podman`) + Dangerzone из исходников/pip; первый запуск тянет контейнер-образ конвертера |
| **Debian / Ubuntu** | Официальный **`.deb`-репозиторий** проекта (`apt install dangerzone`); Podman подтягивается |
| **Fedora** | Официальный RPM-репозиторий (`dnf install dangerzone`) |
| **Arch** (план с июня 2026) | AUR (`dangerzone`); Podman из репозиториев |
| **Windows / macOS** | Готовые установщики с сайта (внутри — свой контейнерный рантайм) |
| **Qubes OS / Tails** | Есть **специальная интеграция** (для этих ОС Dangerzone особенно к месту) |
| **Entware / RT-AX56U** | **N/A** — нужен контейнерный рантайм + рендеринг; на роутере (armv7, 512 МБ) не запустить. Обезвреживай документы на десктопе |
