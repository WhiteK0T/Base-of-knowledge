# 🧩 awesome-osint-chrome-extensions — подборка браузерных расширений для OSINT (Chrome/Brave)

[**awesome-osint-chrome-extensions**](https://github.com/ubikron/awesome-osint-chrome-extensions) (ubikron) — курируемый «awesome»-список **браузерных расширений для Chrome/Brave**, заточенных под **OSINT** (разведку по открытым источникам): анализ сайтов, соцсетей, изображений/видео, геоданных, поиск по никам/email и т.д. Это **не программа и не установщик**, а **указатель ссылок** на расширения в Chrome Web Store (часть совместима и с Firefox-аддонами). ~**100+ расширений** в ~8 категориях. ⚠️ **Лицензии у репозитория нет** (см. ниже), ~230★, обновляется (правки — сегодня).

> [!note] Что это и чего ждать
> Это **карта «что вообще есть»** под браузерный OSINT, а не туториал и не рейтинг. Каждое расширение — **сторонний код от третьего лица**, который получает доступ к страницам и данным браузера. Список лишь собирает ссылки; **проверять безопасность и живость каждого расширения нужно отдельно** (см. предостережение про данные).

## 🧪 Факты против хайпа

> [!caution] Расширения — это сторонний код с доступом к твоим данным
> Сам README предупреждает (цитата): *«есть риск, что расширение передаёт твои персональные данные третьей стороне или делает что-то недокументированное. Не используй расширения в браузерных профилях, где хранятся важные для тебя данные, и проверяй исходники расширений.»*
> - **Заведи отдельный браузерный профиль** (или отдельную ВМ/браузер) под OSINT — без своих логинов/паролей/cookie.
> - Многие OSINT-расширения просят разрешение **«читать и менять все данные на всех сайтах»** — это максимально широкий доступ. Ставь только нужное, читай permissions.
> - Расширение из Web Store может **смениться владельцем и стать malware** при обновлении — периодически ревизуй установленное.

> [!warning] OSINT ≠ вседозволенность — правовые рамки
> Сбор данных из открытых источников законен, но **пробив персональных данных, слежка за конкретным человеком, обход авторизации** — нет.
> - **РФ:** ст. 137 УК (нарушение неприкосновенности частной жизни), 138 (тайна переписки), 272 (неправомерный доступ); сбор/обработка перс. данных — под 152-ФЗ.
> - Легальные сценарии: **свои** активы и бренд-мониторинг, проверка собственного цифрового следа, авторизованный пентест/расследование со scope, CTF, журналистика/исследования в рамках закона.

> [!note] Сверено с источником
> - В посте сказано «Chrome и Brave» — верно (Brave на движке Chromium тянет те же расширения); **часть** пунктов помечена как совместимые с **Firefox**.
> - «Регулярно обновляется» — подтверждается (последняя правда репозитория — сегодня), но **у самого репозитория нет файла лицензии** — формально это «all rights reserved» на текст списка (на сами расширения не влияет, у них свои лицензии).

## 💻 Как это ложится на твои системы

| Система | Применимость |
| :--- | :--- |
| **Gentoo / Debian / Arch** (десктоп) | Основной сценарий. Ставь Chromium/Brave (или Firefox для совместимых) — лучше **отдельный профиль/инстанс** под OSINT. В Gentoo: `www-client/chromium` или `www-client/brave-bin` (overlay); в Debian/Arch — пакет/Flatpak. Расширения — из Chrome Web Store |
| **Entware / RT-AX56U** | **N/A** — это браузерные расширения для десктопа, на роутере браузера нет |

> [!tip] Гигиена OSINT-браузера
> Отдельный профиль + контейнеры вкладок (Firefox Multi-Account Containers) + VPN/прокси, чтобы не светить свой реальный IP при разведке. Логи и находки фиксируй (Hunchly/Forensic OSINT из списка делают это автоматически) — для воспроизводимости расследования.

## 💡 Кому полезно

- **Аналитикам/исследователям/журналистам:** быстрый набор браузерных инструментов под конкретную задачу (соцсети, гео, анализ сайта).
- **Безопасникам:** проверка собственного цифрового следа и поверхности (что видно о твоих доменах/аккаунтах снаружи).
- **Не** замена методологии и **не** разрешение на пробив людей — это лишь каталог инструментов одного слоя.
