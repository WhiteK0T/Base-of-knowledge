# 🪟 GTweak — портативный твикер-деблоатер Windows

[**GTweak**](https://github.com/Greedeks/GTweak) (*Greedeks/GTweak*, **C# / .NET Framework 4.8**, лицензия **BSD-3-Clause**, ~**1.3k★**) — **портативный** (без установки) инструмент «идеальной настройки Windows 10/11»: отключение телеметрии и слежки, удаление UWP-приложений/OneDrive/Edge, отключение служб, настройка интерфейса и питания, очистка кэша/RAM, мониторинг железа. Плюс — спорные функции: **отключение Windows Defender/SmartScreen/UAC/VBS** и **активация Windows (HWID и KMS)**. Поддержка Windows **с 10 (build 18362.116)**.

> [!info] Что внутри (8 разделов)
> - **Утилиты:** сжатие файлов, очистка RAM/кэша.
> - **Конфиденциальность:** отключение телеметрии и «кейлоггеров» Windows/NVIDIA/Intel, задач сбора данных в планировщике, «теневых доменов» Microsoft через `hosts` и фаервол.
> - **Интерфейс:** темы, панель задач, оформление окон.
> - **Приложения:** удаление UWP-пакетов, OneDrive, Edge.
> - **Службы / Система:** отключение «лишних» служб, настройка производительности и плана питания.
> - **Дополнения:** запуск своих скриптов `.ps1/.cmd/.bat/.reg` **с привилегиями TrustedInstaller**.
> - **Инструменты:** мониторинг железа, распаковка файлов.

## 🧪 Факты против хайпа

> [!caution] Отключение Defender/SmartScreen/UAC/VBS — это **ослабление защиты**, а не «оптимизация»
> Tool выключает **Windows Defender, SmartScreen, Antimalware, VBS и UAC**. Это не «ускорение», а сознательное **снятие штатных барьеров безопасности**:
> - **Defender/SmartScreen** — антивирус и репутационный фильтр загрузок; без них вредонос ничто не остановит.
> - **UAC** — запрос на повышение прав; отключение = тихий запуск чего угодно с админскими правами.
> - **VBS / HVCI** — аппаратная изоляция учётных данных (защита от кражи LSASS-хэшей). На современном железе её влияние на FPS обычно **в пределах единиц процентов** — игра свеч сомнительна.
> Делай это **осознанно и только если понимаешь риск**, на изолированной/игровой машине, а не на той, где банк-клиент и почта.

> [!danger] Доверие к самому инструменту — главный риск
> Иронично: деблоатер сам **просит отключить антивирус** («add the program to its exclusions») и при этом исполняет произвольные скрипты с правами **TrustedInstaller** (выше, чем у админа). Это значит:
> - ты отдаёшь программе **полный контроль над системой** — один троянизированный билд = тотальная компрометация (**supply-chain-риск**).
> - Запускай **только** официальные релизы с [GitHub автора](https://github.com/Greedeks/GTweak/releases) (BSD-3, исходники открыты — при паранойе **собери сам** из C#-исходников), сверяй контрольные суммы. **Никаких** сборок «с форумов/телеграма» и «активаторов в комплекте» из левых зеркал.

> [!note] Активация Windows (HWID/KMS) — правовая серая/чёрная зона
> Встроенная «активация» — это **обход лицензирования** Windows (по сути логика в духе Microsoft Activation Scripts). Активация без купленной лицензии — **нарушение лицензионного соглашения Microsoft**, а в коммерческом использовании — и закона (в РФ — ст. 1280/1299 ГК о защите ТСЗАП, плюс лицензионные споры). Здесь я **не привожу инструкций по активации** — это вне «настройки своей системы». Для легальной Windows активация не нужна; для обучения/лабораторных Microsoft даёт **бесплатные оценочные образы** (Evaluation / dev-VM).

> [!tip] Обратимость и здравый смысл
> - Большинство твиков **обратимы**, но не все и не всегда чисто — перед массовыми изменениями сделай **точку восстановления** и (лучше) полный образ диска.
> - «Удаление Edge/OneDrive» и отключение служб иногда **ломает** обновления, Магазин, WSL2, Hyper-V, песочницу. Меняй **по одному**, проверяя, что нужное ещё работает.
> - На свежем железе ощутимого прироста производительности от деблоата чаще **нет** — основная польза это приватность и чистота системы, а не FPS.

## 💻 Как это ложится на твои системы

GTweak — **исключительно для Windows** (нужен .NET Framework 4.8, WinAPI/реестр). На основных системах владельца (Linux) он не запускается и не нужен — Linux не требует «деблоата».

| Система | Применимость |
| :--- | :--- |
| **Windows 10/11** (нативно / дв.загрузка) | Целевая платформа. Портативный `.exe`, нужны права администратора |
| **Windows в ВМ на Linux** | Безопасный способ попробовать: подними Windows в QEMU/KVM/VirtualBox, делай снапшот **до** твиков и катай эксперименты без риска для основной ОС |
| **Gentoo / Debian / Arch** (десктоп Linux) | **N/A** — нативно не запускается; через Wine маловероятно (TrustedInstaller/драйвер/реестр). Аналог идеи «убрать лишнее» в Linux — просто не ставить лишнего |
| **Entware / RT-AX56U** | **N/A** — это не Windows и не десктоп |

## 💡 Кому полезно

- **Тем, кто настраивает чистую Windows** под себя (приватность, удаление UWP-хлама) и **понимает**, что отключает.
- **Не** новичкам «нажать всё подряд для ускорения»: отключение Defender/UAC/VBS и удаление компонентов легко превращается в дырявую или сломанную систему.
- Перед использованием — **точка восстановления/образ**, запуск только официального билда.
