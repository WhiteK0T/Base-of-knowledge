# 🪟 System Informer — мониторинг процессов и охота на малварь (преемник Process Hacker)

[**System Informer**](https://github.com/winsiderss/systeminformer) — бесплатный мощный инструмент для **Windows**: мониторинг ресурсов системы, отладка ПО и **обнаружение вредоносов**. Это **прямой преемник Process Hacker** (тот же проект под управлением Winsider Seminars & Solutions). Язык — C, лицензия **MIT**, ~15.1k★, портативный (без установки). Грубо говоря — это «Диспетчер задач на стероидах».

> [!info] Это бывший Process Hacker
> Если искал «Process Hacker» — это он и есть, переименованный и развиваемый дальше. Старое имя живёт в topic-теге и драйвере. Функционал — надмножество Диспетчера задач и Sysinternals Process Explorer.

## ✨ Возможности

- **Обзор активности системы** в реальном времени: графики и статистика CPU/RAM/диска/сети с подсветкой изменений.
- **Процессы:** кто сколько ест ресурсов, дерево процессов, подробные свойства, токены/привилегии, обработка приоритетов, завершение «неубиваемых» процессов.
- **Сеть:** активные сетевые соединения с привязкой к процессу.
- **Диск:** какие процессы и к каким файлам обращаются в реальном времени (как Resource Monitor, но детальнее).
- **Стек-трейсы:** включая **kernel mode, WOW64 и .NET** — удобно для отладки и анализа.
- **Службы (services):** создание, редактирование, запуск/остановка/удаление.
- **Поиск handles/DLL:** найти, какой процесс держит файл/ключ реестра.

> [!tip] Для анализа малвари
> Подсветка новых/завершающихся процессов, просмотр загруженных модулей, цифровых подписей, строк в памяти, сетевых соединений процесса и принудительное завершение защищённых процессов делают его рабочим инструментом для **триажа подозрительной активности**. Многое требует прав **администратора**.

> [!warning] Драйвер режима ядра (KSystemInformer)
> Часть «продвинутых» функций (доступ к защищённым процессам, низкоуровневые операции) работает через **драйвер уровня ядра**. Это нормально для такого инструмента, но: ставь сборки **только с официального репозитория/релизов**, антивирусы иногда реагируют на такой инструмент (как и на Process Hacker — его детектят как «riskware/hacktool» из-за мощности, это **ложноположительно** для оригинальной сборки). Не качай «System Informer» со сторонних сайтов.

## 📦 Установка

| Система | Как |
| :--- | :--- |
| **Windows 10/11** (32/64-bit) | портативный exe с [официального сайта/Releases](https://github.com/winsiderss/systeminformer/releases) (стабильные + canary-сборки). Есть в **winget**: `winget install SystemInformer`, и в Chocolatey/Scoop. Настройки хранятся в XML рядом |
| **Сборка из исходников** | Visual Studio 2022+ (Windows) |
| **Linux (Gentoo/Debian/Ubuntu/Arch)** | **N/A** — Windows-инструмент. Аналоги под Linux: `htop`/`btop`/`System Monitor`, `lsof`/`ss` для файлов и сокетов, `strace` для трейсов |
| **Entware / RT-AX56U** | **N/A** — для роутера по SSH: `top`/`htop`, `netstat`/`ss`, `lsof` |

> [!note] Версии
> Стабильный релиз — ветка **3.x** (например `v3.2.x`, май 2025), плюс регулярные **canary/nightly**-сборки с новыми фичами. Для боевого использования бери стабильную; canary — если нужны самые свежие возможности.

## 💡 Кому полезно

- **Windows-пользователю/админу**, которому мало стандартного Диспетчера задач: глубже видно процессы, службы, сеть, диск, handles.
- **Анализ инцидентов/малвари**: что за процесс, кто его породил, какие соединения и файлы держит, чем подписан.
- **Разработчику** — стек-трейсы (kernel/WOW64/.NET), отладка зависаний и утечек handle.
