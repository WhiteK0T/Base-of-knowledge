# ✏️ OpenPencil — опенсорсный ИИ-редактор дизайна (альтернатива Figma, открывает `.fig`)

**OpenPencil** ([github.com/open-pencil/open-pencil](https://github.com/open-pencil/open-pencil), [openpencil.dev](https://openpencil.dev/ru/)) — **local-first опенсорсный редактор дизайна**, позиционируемый как **AI-native альтернатива Figma**. Главная фишка — **нативно открывает и редактирует `.fig`-файлы** Figma (через бинарный кодек **Kiwi**), работает **полностью локально** (без аккаунта/сервера/интернета), несёт **встроенного ИИ-агента** (90+ инструментов) и **MCP-сервер** для связки с [Claude Code](Claude%20Code%20%E2%80%94%20гайд.md)/Cursor/Windsurf. Стек — **Tauri** (TypeScript + Vue + Rust, рендер на **Skia/WebGPU**), лицензия **MIT**, ~6.1k★, версия **v0.13.2** (30.05.2026).

> [!warning] Факты против поста: это альтернатива Figma, а не «дешёвый Claude Design»
> Пост подаёт его как «Claude Design за 0 рублей». Уточняем:
> - OpenPencil — **редактор-холст в духе Figma/Penpot** (двигаешь слои, рисуешь, открываешь `.fig`), а **не** генератор макетов из промпта, как Claude Design. Это **разные классы** инструментов. Если нужен именно «опиши — получи дизайн», ближе [Open Design](Open%20Design%20%E2%80%94%20опенсорсная%20альтернатива%20Claude%20Design%20%28скиллы%20%2B%20дизайн-системы%29.md).
> - «Бесплатно навсегда» — правда (MIT, локально, без подписки). Но встроенный ИИ-агент работает через **твою модель/ключ** (или связку по MCP с твоим Claude Code/Cursor) — токены всё равно твои.
> - **Молодой проект (0.x)**: импорт **проприетарного** `.fig` реверс-инжинирен — на сложных макетах возможны расхождения в точности. Для боевого пайплайна проверяй результат.

## ✨ Что умеет

- **Совместимость с Figma:** открывает/редактирует `.fig` нативно (кодек **Kiwi** для точного сохранения данных), copy-paste-интероп, поддержка **Figma Plugin API**.
- **Встроенный ИИ-агент:** чат с **90+ инструментами** — создание фигур, стилизация, работа с лейаутом, анализ дизайн-токенов.
- **Экспорт в код:** **Tailwind CSS** и **JSX** из макетов (то, ради чего обычно и берут такие тулзы — мост дизайн→фронтенд).
- **MCP-сервер:** подключается к [Claude Code](Claude%20Code%20%E2%80%94%20гайд.md), Cursor, Windsurf — агент из другого репозитория читает/правит проект как структурированный API (см. [MCP](MCP%20%E2%80%94%20серверы%20Model%20Context%20Protocol.md)).
- **P2P-совместная работа:** реалтайм через **WebRTC без сервера** (прямое соединение между участниками).
- **Headless CLI:** инспекция `.fig`, экспорт, анализ, **JSON-вывод для CI**.

> [!info] Чем отличается от соседней заметки [Open Design](Open%20Design%20%E2%80%94%20опенсорсная%20альтернатива%20Claude%20Design%20%28скиллы%20%2B%20дизайн-системы%29.md)
> - **OpenPencil** = **редактор-холст** (как Figma): ты сам двигаешь слои + ИИ-помощник внутри. Открывает `.fig`.
> - **Open Design** = **слой поверх твоих кодинг-агентов** (Claude Code/Codex…), генерирует артефакты по описанию из готовых дизайн-систем; своего холста-редактора Figma-типа нет.
> Они дополняют друг друга, а не дублируют.

## 📦 Установка

Десктоп — на **Tauri**, поэтому сборки есть под все ОС (в релизах: `.AppImage`/`.deb`/`.rpm` для Linux, `.dmg` для macOS, `.msi`/`.exe` для Windows, x64 и arm64). Есть и **веб-версия** (~7 МБ) на [app.openpencil.dev](https://app.openpencil.dev) — без установки, прямо в браузере (нужен WebGPU).

| Система владельца | Как ставить |
| :--- | :--- |
| **Gentoo** (основная) | официального ebuild нет → бери **AppImage** из [Releases](https://github.com/open-pencil/open-pencil/releases) (`OpenPencil_*_amd64.AppImage`, `chmod +x` и запуск) или собирай из исходников (Tauri: нужны `rust`, `nodejs`+`pnpm`, `webkit2gtk`). Либо просто **веб-версия** |
| **Debian / Ubuntu** | `.deb` из релизов: `sudo apt install ./OpenPencil_0.13.2_amd64.deb` (или AppImage) |
| **Arch** (план с июня 2026) | AppImage из релизов, либо собрать из AUR/исходников (Tauri-зависимости); `.rpm` не подходит |
| **macOS / Windows** | `.dmg` (mac, в т.ч. aarch64) или `.msi`/`.exe` (Win, x64/arm64); на macOS также через **Homebrew** |
| **Entware / RT-AX56U** | **N/A** — это десктоп-GUI на Skia/WebGPU, тяжёлый рендер; на роутер не ставится |

> [!tip] Самый быстрый способ попробовать
> Не хочешь ничего ставить — открой **веб-версию** [app.openpencil.dev](https://app.openpencil.dev) в браузере с WebGPU (свежие Chromium/Firefox) и закинь свой `.fig`.

## 💡 Кому полезно

- Хочешь **открыть/поправить `.fig`** без аккаунта Figma и облака, **локально и бесплатно**.
- Нужен мост **дизайн → код** (экспорт в **Tailwind/JSX**) и связка с [Claude Code](Claude%20Code%20%E2%80%94%20гайд.md)/Cursor по **MCP**, чтобы агент правил макет.
- Нужна **P2P-коллаборация** без поднятия сервера.
- **Учитывай зрелость:** проект на ранней стадии (0.x), импорт чужого формата `.fig` может быть неидеальным — для критичных макетов сверяйся с оригиналом.
