# AI Website Cloner Template

[**ai-website-cloner-template**](https://github.com/JCodesMore/ai-website-cloner-template) (автор — JCodesMore, лицензия **MIT**) — это переиспользуемый **шаблон-пайплайн** для реверс-инжиниринга любого сайта в чистый, современный код на **Next.js**. Вы указываете URL, запускаете команду `/clone-website`, и AI-агент сам исследует страницу, извлекает дизайн-токены и ассеты, пишет спецификации компонентов и параллельно пересобирает каждую секцию.

> [!info] Это не отдельный агент
> Шаблон не содержит своей модели и не является самостоятельным агентом — это **набор инструкций/скилл поверх существующих AI-агентов** (Claude Code, Codex CLI и др.). Связанные инструменты см. в [Сводной таблице AI-агентов](Сводная%20таблица%20AI-агентов%20для%20программирования%20%28июнь%202026%29.md).

Вышел в марте 2026 и быстро набрал ~13–16k звёзд на GitHub, превратив «vibe cloning» из мема в воспроизводимый инженерный процесс.

## ⚙️ Технологический стек

- **Next.js 16** (App Router, React 19, TypeScript strict)
- **shadcn/ui** (примитивы Radix + Tailwind CSS v4)
- **Tailwind CSS v4** с дизайн-токенами в формате `oklch`
- **Lucide React** — иконки по умолчанию

## 🔄 Пайплайн `/clone-website`

Команда запускает многофазный конвейер:

1. **Reconnaissance (разведка)** — скриншоты, извлечение дизайн-токенов, прогон по интерактивным состояниям.
2. **Foundation (фундамент)** — обновляет шрифты, цвета, глобальные стили, скачивает все ассеты.
3. **Component Specs (спецификации)** — пишет детальные спеки с точными вычисленными значениями CSS, состояниями, поведением и контентом.
4. **Parallel Build (параллельная сборка)** — раздаёт задачи builder-агентам, каждый в своём **git worktree** на отдельную секцию.
5. **Assembly & QA (сборка и проверка)** — сливает worktree, собирает страницу и прогоняет визуальные diff-сравнения.

## 🤖 Поддерживаемые агенты

Claude Code (**рекомендуется** Opus — в README указан 4.7; актуальные Opus 4.8 тоже подходят), Codex CLI, OpenCode, GitHub Copilot, Cursor, Windsurf, Gemini CLI, Cline, Roo Code, Continue, Amazon Q, Augment Code, Aider. Инструкции для каждого — в файле `AGENTS.md` репозитория.

## 🚀 Установка и запуск

1. Нажать на GitHub кнопку **«Use this template»** и создать свой репозиторий.
2. Склонировать его локально.
3. `npm install`.
4. Запустить агента (`claude --chrome` или эквивалент вашего агента).
5. Выполнить `/clone-website <target-url>`.
6. Доработать результат под себя.

**Требования:** Node.js **24+** и любой из поддерживаемых AI-агентов.

> [!warning] Юридический момент
> Автор прямо **запрещает** использование шаблона для фишинга, выдачи себя за других и обмана. Нельзя присваивать чужие дизайны, логотипы и брендовые ассеты. Многие сайты **запрещают скрейпинг** в своих условиях использования (ToS) — это нужно проверять заранее. Инструмент уместен для прототипирования, обучения и работы со своими/разрешёнными ресурсами.
