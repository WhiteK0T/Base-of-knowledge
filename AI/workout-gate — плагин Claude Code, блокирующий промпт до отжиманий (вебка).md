# 💪 workout-gate — плагин Claude Code, блокирующий промпт до отжиманий (вебка)

**workout-gate** ([github.com/BotchetDig/workout-gate](https://github.com/BotchetDig/workout-gate)) — шуточно-полезный **хук для [Claude Code](Claude%20Code%20%E2%80%94%20гайд.md)**: периодически замораживает ввод промпта, включает **веб-камеру** и не пускает дальше, пока ты не сделаешь отжимания или приседания — счёт повторов идёт **вживую по камере**. Девиз: «Your AI works hard, so should you». MIT, ~127★, Python. Распознавание поз — **локально** через MediaPipe, без облака.

> [!warning] Отделяем факты от подачи поста
> | Заявление поста | Реально |
> | :--- | :--- |
> | «раз в час останавливает работу» | по умолчанию — **каждые 15 промптов**; почасового нет, но есть **тайм-режим** (напр. 30 мин) и «рулетка» (10% шанс) — всё настраивается |
> | «заставит качаться» | **fail-open**: нет вебки/краш — промпт проходит; это опенсорс-хук, отключается. Жёсткой блокировки нет |
> | «врубает камеру» (приватность?) | да, но обработка **полностью локальная** (MediaPipe Pose), кадры никуда не уходят |

## 🧬 Как работает

- Хук **`UserPromptSubmit`** считает промпты. При срабатывании: считает случайное число повторов → **пишет «долг» на диск** (закрыть вкладку и сбежать не выйдет) → открывает окно вебки и **замораживает промпт** → отпускает после выполнения.
- **Детекция (MediaPipe Pose, локально):** отжимания — по углу локтя (профиль, горизонтальное тело); приседания — по углу колена (анфас, вертикально). 1 повтор = полный сход + полное разгибание со сглаживанием и проверкой позы.
- **Долг персистентный**: бросил на середине — недоделанные повторы сохранятся атомарно по-повторно и перейдут в следующую сессию.
- Своё упражнение — подклассом `ExerciseCounter` в `detector.py`.

## ⚙️ Настройка

| Параметр | Варианты |
| :--- | :--- |
| **Частота** | каждые N промптов (деф. 15) · по времени (30 мин) · рулетка (10%) |
| **Повторы** | под тебя — 25–50% от заявленного «макс за подход» |
| **Пресеты** | `chill` (кажд. 25, 3–6) · `demo` (каждый, 5–8) · `hardcore` (кажд. 5, 15–25) |

```bash
! workout set freq 15            # частота — каждые 15 промптов
! workout set reps squats 8 15   # приседания: от 8 до 15 повторов
! workout                        # дашборд (веб + терминал): стрик, графики за 7 дней
```

## 📦 Установка

Нужны **Python 3.9–3.13**, **вебка** и интернет на первом запуске (тянет модель поз MediaPipe/OpenCV ~9 МБ). Установщик создаёт venv, ставит зависимости и проводит 30-секундный мастер (спросит твой «макс за подход», даст выбрать триггер, прогонит тест на 2 отжимания).

```bash
# Вариант 1 — как плагин Claude Code:
/plugin marketplace add BotchetDig/workout-gate
/plugin install workout-gate@workout-gate
```

```bash
# Вариант 2 — однострочный установщик (macOS / Linux):
curl -fsSL https://raw.githubusercontent.com/BotchetDig/workout-gate/main/get.sh | bash
```

```bash
# Вариант 3 — вручную:
git clone https://github.com/BotchetDig/workout-gate.git && cd workout-gate
./install.sh
```

| Система | Как ставить |
| :--- | :--- |
| **Gentoo / Debian-Ubuntu / Arch** | `git` + `python3` (3.9–3.13) на PATH, затем плагин/`get.sh`; нужна рабочая вебка |
| **macOS** | zero-config (рекомендованная авторами) |
| **Entware / RT-AX56U** | **неприменимо** — нет Claude Code/вебки, новелти-инструмент для десктопа |

> [!caution] Камера + `curl … | bash`
> Тул работает с **веб-камерой** (обработка локальная, MediaPipe — кадры не уходят в сеть, но включение камеры само по себе учитывай). Установщик одной строкой исполняет скачанный скрипт — доверяешь репо, ок; иначе глянь `get.sh`/`install.sh` или ставь вручную.

## 💡 Когда полезно

- Сидишь в [Claude Code](Claude%20Code%20%E2%80%94%20гайд.md) часами и хочешь **принудительные микро-паузы** на разминку — «beneficial friction».
- Заходит геймификация (стрики, графики) и не смущает работа с камерой.
- **Не для прод-окружения/совместных репо** — это новелти-хук; на сервере/CI/без вебки бессмысленно (там он просто fail-open пропускает).
