# 🗺️ Scan (Foglamp) — кодовую базу в интерактивный граф руками твоего же агента

**Scan** ([foglamp.dev/scan](https://www.foglamp.dev/scan)) от команды **Foglamp** — сервис, который превращает репозиторий в **интерактивную карту (граф)**: интеграции, методы, функции, переменные, обращения к БД и прочие части кода становятся **нодами**, связи между ними рисуются на карте, любой узел можно приближать, а готовой картой — **делиться по ссылке в один клик**. Работает поверх [Claude Code](Claude%20Code%20%E2%80%94%20%D0%B3%D0%B0%D0%B9%D0%B4.md) и Codex. **Установка и аккаунт не нужны** — карту строит твой уже настроенный агент по одному промпту.

> [!warning] Отделяем факты от рекламы поста
> Пост обещает «разберёшься в любом коде за 5 секунд». Сверка с сайтом:
> | Заявление | Реально |
> | :--- | :--- |
> | «инструмент за 5 секунд разбирает любой код» | карту строит **твой агент** (Claude Code/Codex) — это его проход по репозиторию, **твои токены и время**, а не мгновенная магия сервиса |
> | «без установки и аккаунта» | правда для запуска: даёшь агенту промпт — он сканирует локально. Но **готовая карта публикуется на серверах Foglamp** (иначе делиться нечем) |
> | «поймёт любой код» | структуру агент берёт разбором репозитория, смыслы — LLM: на больших/незнакомых проектах **возможны ошибки и домыслы**, перепроверяй |
> | «поделиться в один клик» | да — но см. приватность ниже |

> [!caution] Приватность — граф уходит на сторонний сервер
> «Поделиться картой по ссылке» означает, что **структура твоего кода** (имена функций, модулей, таблиц БД, внешних интеграций) **хостится у Foglamp**. Для учебного/публичного репозитория — ок. Для **закрытого/рабочего кода** это утечка архитектуры на третью сторону: сам код, может, и не загружается целиком, но карта раскрывает устройство проекта. Не делай публичные карты по приватным репо без разрешения.

## 🧩 Что показывает на карте

- **Ноды** — интеграции, методы, функции, переменные, обращения к БД и другие части кода.
- **Связи** между частями — наглядно, стрелками на карте (кто что вызывает/использует).
- **Зум** — любой узел приближается для детального разбора.
- **Шеринг** — карта отдаётся по ссылке одним кликом.
- Годится не только для кода, но и для **разбора работы ИИ-агентов** (по заявлению авторов).

## ⚙️ Как это работает (модель «агент делает сам»)

Ключевая идея: Scan — **не отдельная программа и не standalone-анализатор**, а связка «промпт + хостинг карты». Твой AI-агент ([Claude Code](Claude%20Code%20%E2%80%94%20%D0%B3%D0%B0%D0%B9%D0%B4.md) или Codex) по инструкции проходит репозиторий, собирает граф и публикует его как интерактивную карту на стороне Foglamp. Отсюда — «без установки»: вся тяжёлая работа выполняется уже стоящим у тебя агентом.

## 🔁 Scan vs [Understand Anything](Understand%20Anything%20%E2%80%94%20%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%B0%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9%20%D0%B3%D1%80%D0%B0%D1%84%20%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%BF%D0%BE%20%D0%BA%D0%BE%D0%B4%D0%BE%D0%B2%D0%BE%D0%B9%20%D0%B1%D0%B0%D0%B7%D0%B5.md) — почти одно и то же

Обе идеи из одного канала и решают одну задачу — **кодовая база → интерактивный граф руками твоего агента**. Разница в упаковке:

| | **Scan (Foglamp)** | **[Understand Anything](Understand%20Anything%20%E2%80%94%20%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%B0%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9%20%D0%B3%D1%80%D0%B0%D1%84%20%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%BF%D0%BE%20%D0%BA%D0%BE%D0%B4%D0%BE%D0%B2%D0%BE%D0%B9%20%D0%B1%D0%B0%D0%B7%D0%B5.md)** (Egonex-AI) |
| :--- | :--- | :--- |
| Что это | веб-сервис + промпт, карта **хостится онлайн** | **плагин** к агенту, граф `.understand-anything/knowledge-graph.json` **локально** |
| Хосты | Claude Code, Codex | Claude Code (нативно), Cursor, Copilot, Codex, Gemini CLI и др. |
| Установка | не нужна (промпт) | ставится плагином/скриптом |
| Открытость | закрытый веб-сервис | открытый (MIT), исходники на GitHub |
| Шеринг | ✅ ссылка в один клик | локально (делишься файлом/дашбордом сам) |
| Приватность | карта на серверах Foglamp | всё остаётся у тебя |

Вывод: **Scan** удобнее, когда надо быстро *показать* карту кому-то (ссылка); **Understand Anything** — когда важны локальность/приватность и открытый код.

## 🖥️ Применимость на системах владельца

| Система | Применимо |
| :--- | :--- |
| **Gentoo / Debian-Ubuntu / Arch** | ✅ — везде, где стоит [Claude Code](Claude%20Code%20%E2%80%94%20%D0%B3%D0%B0%D0%B9%D0%B4.md) или Codex; сам Scan ничего локально не ставит |
| **Entware / RT-AX56U** | ❌ неприменимо — десктоп-dev-инструмент над AI-агентом, на роутере агент не гоняют |

## 💡 Когда полезно

- **Быстро показать структуру** незнакомого/своего проекта коллеге — расшарить карту ссылкой.
- Визуально пройтись по связям функций/модулей/БД, не читая всё вручную.
- Помни: карту рисует **твоя модель** (расход токенов), смысловые подписи стоит перепроверять, а для приватного кода — **не публиковать** карту наружу.
- Нужна локальность и открытый код — бери [Understand Anything](Understand%20Anything%20%E2%80%94%20%D0%B8%D0%BD%D1%82%D0%B5%D1%80%D0%B0%D0%BA%D1%82%D0%B8%D0%B2%D0%BD%D1%8B%D0%B9%20%D0%B3%D1%80%D0%B0%D1%84%20%D0%B7%D0%BD%D0%B0%D0%BD%D0%B8%D0%B9%20%D0%BF%D0%BE%20%D0%BA%D0%BE%D0%B4%D0%BE%D0%B2%D0%BE%D0%B9%20%D0%B1%D0%B0%D0%B7%D0%B5.md); полный список агентов — в [сводной таблице](%D0%A1%D0%B2%D0%BE%D0%B4%D0%BD%D0%B0%D1%8F%20%D1%82%D0%B0%D0%B1%D0%BB%D0%B8%D1%86%D0%B0%20AI-%D0%B0%D0%B3%D0%B5%D0%BD%D1%82%D0%BE%D0%B2%20%D0%B4%D0%BB%D1%8F%20%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B8%D1%80%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F%20%28%D0%B8%D1%8E%D0%BD%D1%8C%202026%29.md).
