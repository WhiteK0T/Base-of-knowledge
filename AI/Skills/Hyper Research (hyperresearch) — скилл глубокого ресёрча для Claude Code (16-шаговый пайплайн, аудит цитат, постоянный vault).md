# 🔬 Hyper Research (hyperresearch) — скилл глубокого ресёрча для Claude Code

**hyperresearch** ([github.com/jordan-gibbs/hyperresearch](https://github.com/jordan-gibbs/hyperresearch)) — «harness глубокого исследования» для [Claude Code](../Claude%20Code%20%E2%80%94%20%D0%B3%D0%B0%D0%B9%D0%B4.md): ставится как скилл, запускается командой `/hyperresearch <тема>` и прогоняет **16-шаговый пайплайн** — от разбиения темы на «матрицу покрытия» до состязательной вычитки — а на выходе даёт **отчёт с проверенными цитатами и полным списком источников**. Каждый прочитанный источник складывается в **постоянный локальный vault (Markdown + SQLite)**, который переиспользуется в следующих сессиях. Python 3.11–3.13, **MIT, ~1250★**, автор — Jordan Gibbs; проект живой (обновляется).

> [!warning] Отделяем факты от рекламы поста
> Пост обещает, что ИИ «откопает ЛЮБУЮ инфу», и называет скилл «имбовейшим». Сверка с репозиторием:
> | Заявление | Реально |
> | :--- | :--- |
> | «откопает ЛЮБУЮ инфу» | ищет только то, что **доступно веб-поиску/фетчу агента**: за пейволлом, не проиндексированное, офлайновое или свежее «вчерашнее» он может **не найти**. Это мощный harness, не всеведение |
> | «делит задачу на матрицы покрытия и сопоставляет источники» | ✅ так и есть: decompose → coverage matrix, граф противоречий, кластеризация дублей, сверки |
> | «даёт только достоверный результат» | есть **реальные механизмы против галлюцинаций** (ниже), но 100 % достоверности не гарантирует никто — итог всё равно **проверяй сам** |
> | «лидер бенчмарка DeepResearch-Bench» | по README это **внутренний/прогнозный** замер («benchmarked internally», «forward-looking projection», «third-party validation is pending») — то есть **самозаявка**, а не независимо подтверждённое первое место |

> [!caution] Это не бесплатно в работе (хотя код — MIT)
> Сам скилл открыт (MIT), но **гоняет он твой Claude Code**: один прогон — это **250+ источников**, 16 шагов и куча параллельных субагентов; режим «диссертация» — **25–80 тыс. слов и 300–450 источников**. Это **много токенов и денег/лимитов**. Нужен доступ Claude Code + веб-поиск; на слабом тарифе большой ресёрч будет дорогим и долгим. Для коротких запросов есть авто-режим «light» (5 шагов).

## 🧩 Как устроено (почему это не «промпт на удачу»)

Входной скилл — тонкий роутер: он фиксирует каноничную формулировку запроса и вызывает **по одному под-скиллу на фазу** через `Skill`-механику Claude Code. Процедура шага грузится в контекст **только когда шаг реально исполняется** — так длинный пайплайн не «теряет» шаги по мере замусоривания контекста. Ключевые узлы:

- **Decompose + coverage matrix** — тема → атомарные пункты + матрица покрытия + классификация по «ярусу» сложности.
- **Width sweep** — план многоперспективного поиска и параллельные волны фетчеров.
- **Граф противоречий** — пары противоречий по корпусу сводятся в ранжированные кластеры.
- **Independence audit** — «синдикация ≠ консенсус»: пять перепечаток одного пресс-релиза считаются **как один** источник, а не пять.
- **Cite-check** — скептический аудитор проверяет, что **каждая цитата реально подтверждает** предложение; выдуманные цитаты/незамеченные ретракции — жёсткий блок.
- **4 состязательных критика** параллельно атакуют черновик; правки вносит **tool-locked патчер** (только точечные Edit — он физически не может переписать отчёт).
- **Resume** — упавший прогон возобновляется с точного шага (манифест).

## 📤 Что на выходе

- `final_report.md` — отчёт со **сквозной провенанс-привязкой** (каждое утверждение → источник).
- **Постоянный vault** (Markdown + SQLite) со всеми источниками — следующая сессия сначала ищет в нём, потом лезет в сеть.
- Масштаб от **быстрых 30-минутных** справок до **диссертации на главы** (opt-in).

## 🛠️ Установка

```bash
cd твой-проект
pip install hyperresearch && hyperresearch install
# затем в Claude Code:
/hyperresearch <любой запрос>
```

Python **3.11–3.13** (3.14 пока не поддерживается — ставь 3.13 через `pyenv`/`uv`). `hyperresearch install --global` делает `/hyperresearch` доступным из любой сессии Claude Code ценой ~15 строк в системном напоминании каждой сессии; **per-project** установка (без `--global`) держит прочие сессии чистыми.

## 🖥️ Применимость на системах владельца

Это Python-CLI + скилл поверх Claude Code — нужен **десктоп с Claude Code, Python 3.11–3.13 и сетью**:

| Система | Как использовать |
| :--- | :--- |
| **Gentoo / Debian-Ubuntu / Arch** | ✅ ставишь Python 3.11–3.13 (у Gentoo — целевой слот; у Arch свежий Python держи в `venv`/`uv`, т.к. системный может уйти на 3.14) + Claude Code → `pip install hyperresearch` |
| **Entware / RT-AX56U** | ❌ нереально: многоагентный ресёрч на 512 МБ armv7 не тянуть (и Node-агент, и Python-оркестрация тяжёлые). Гоняй на десктопе/VPS, роутер тут ни при чём |
