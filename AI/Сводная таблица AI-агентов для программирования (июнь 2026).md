## 📖 Описание каждого агента

### **Claude Code**
*Разработчик:* Anthropic | *Лицензия:* Проприетарная  
Мощный агент, работающий через CLI или IDE, который самостоятельно анализирует код, вносит изменения, запускает тесты и итеративно решает сложные задачи. Тесно интегрирован с моделями Anthropic (Claude 4.x — Opus 4.5/4.8, Sonnet, Haiku). На середину 2026 — один из лидеров по бенчмаркам «тяжёлого» рефакторинга (Opus достигает ~80% на SWE-bench Verified). Идеален для рефакторинга, многофайловых правок и отладки.  
*Цена:* от $20/мес.

### **OpenCode**
*Разработчик:* Сообщество (SST) | *Лицензия:* MIT  
Гибкий агент с открытым исходным кодом, поддерживающий 75+ LLM-провайдеров (включая локальные модели). Умеет работать с LSP (языковые серверы) и обеспечивает полную приватность. Доступен как CLI, десктопное приложение и расширение для IDE.  
*Цена:* бесплатно (BYOK).

### **Aider**
*Разработчик:* Сообщество | *Лицензия:* Apache 2.0  
Легковесный CLI-агент с глубокой интеграцией с Git. Автоматически создаёт коммиты, поддерживает чат с историей, может редактировать несколько файлов одновременно. Позволяет переключаться между разными моделями (OpenAI, Anthropic, Groq, локальные).  
*Цена:* бесплатно (BYOK).

### **Cline**
*Разработчик:* Сообщество | *Лицензия:* Apache 2.0  
Расширение для VS Code. Выполняет сложные задачи: создаёт/редактирует файлы, выполняет команды в терминале, управляет браузером для сбора информации. Может работать в полностью автономном режиме.  
*Цена:* бесплатно (BYOK).

### **Roo Code**
*Разработчик:* Roo (Сообщество) | *Лицензия:* Apache 2.0  
Расширение для VS Code, **форк Cline** (бывший «Roo Cline»). Делит с Cline ~90% «мозга», но развивается агрессивнее и добавляет свои фишки: **кастомные режимы-роли** (Architect / Code / Ask и собственные), diff-правки (экономия ~30% токенов), мульти-агентные персоны, более широкий выбор моделей. Бэкенда нет — код идёт напрямую к провайдеру (Anthropic, OpenAI и т.д.).  
*Цена:* бесплатно (BYOK).  
> [!warning] Статус: расширение свёрнуто (15 мая 2026)
> При 3 млн установок команда свернула расширение, Roo Code Cloud и Router, переключившись на облачный агент **Roomote**. Архивный бинарник продолжает работать, но без обновлений. Официально рекомендованные замены — **Cline** (апстрим, `.roorules` → `.clinerules`) и **Kilo Code** (сохраняет больше фич, включая Orchestrator-режим).

### **Kilo Code**
*Разработчик:* Kilo (Сообщество) | *Лицензия:* Apache 2.0  
Открытое расширение для VS Code, фактически **суперсет Roo Code и Cline** — вобрало лучшее из обоих в один инструмент. Особенности: marketplace режимов и правил, **Orchestrator** для разбиения задач на под-агенты, поддержка 500+ моделей через BYOK, богатая интеграция с VS Code и визуальный review. GA состоялся 2 апреля 2026; после закрытия Roo Code — один из основных преемников.  
*Цена:* бесплатно (BYOK).

### **Continue**
*Разработчик:* Continue | *Лицензия:* Apache 2.0  
Фреймворк для встраивания AI-агентов в VS Code и JetBrains IDE. Позволяет создавать собственных агентов под специфические задачи (документирование, генерация тестов, рефакторинг).  
*Цена:* бесплатно (BYOK).

### **Gemini CLI**
*Разработчик:* Google (Open source) | *Лицензия:* Apache 2.0  
Официальный CLI-агент от Google, оптимизированный для моделей Gemini. Поддерживает режим «автоисследования» — может самостоятельно выполнять цепочки действий, изучать код, искать информацию. Есть бесплатный лимит, после — BYOK.

### **Codex CLI**
*Разработчик:* OpenAI | *Лицензия:* Open source  
Официальный CLI-агент от OpenAI. Работает с моделями линейки GPT-5.x Codex (лидер Terminal-Bench 2.0). Позволяет выполнять команды, редактировать файлы, отвечать на вопросы о кодовой базе. Есть и облачный режим (Codex Cloud) для фоновых задач.  
*Цена:* BYOK (оплата API OpenAI).

### **Cursor**
*Разработчик:* Anysphere | *Лицензия:* Проприетарная  
Полноценная IDE на основе VS Code со встроенным AI-агентом. Умеет предсказывать следующие правки (как Copilot, но умнее), может рефакторить весь проект по естественноязыковому запросу. Популярен среди стартапов.  
*Цена:* от $20/мес.

### **GitHub Copilot**
*Разработчик:* GitHub/Microsoft | *Лицензия:* Проприетарная  
Самый массовый AI pair programmer. В последних версиях добавлен «агентный» режим, который может не просто дописывать строки, а выполнять многошаговые задачи в коде.  
*Цена:* от $10/мес.

### **Windsurf**
*Разработчик:* Cognition (ранее Codeium) | *Лицензия:* Проприетарная  
Собственная IDE с упором на скорость работы и удобный UX. Агент (Cascade) хорошо понимает контекст всего проекта, умеет самостоятельно исправлять баги и писать boilerplate-код. В декабре 2025 продукт куплен **Cognition** (создатели Devin) за ~$250 млн; появилась собственная быстрая модель **SWE-1.5** и функция **Codemaps**, а Devin встроен прямо в IDE.  
*Цена:* ~$15/мес.

### **Trae**
*Разработчик:* ByteDance | *Лицензия:* Проприетарная  
AI-native IDE на базе VS Code (macOS/Windows + облачная Cloud IDE). Сильные стороны — низкая цена и автономный **Builder / SOLO-режим**: по описанию на естественном языке генерирует проект из нескольких файлов с живым превью; умеет собирать React-компоненты по скриншоту из Figma и делегировать тесты/линтинг под-агентам. Доступ к Claude, GPT, Gemini, DeepSeek. **Важно:** телеметрия ByteDance и длительное хранение данных — есть режим Privacy Mode, но стоит учитывать при работе с чувствительным кодом.  
*Цена:* бесплатный план; Lite — $3/мес, Pro — $10/мес.

### **Amazon Q**
*Разработчик:* AWS | *Лицензия:* Проприетарная  
CLI-агент для разработчиков, работающих в AWS. Помогает писать инфраструктурный код (CloudFormation, CDK), отлаживать Lambda, искать уязвимости в security groups. Глубоко интегрирован с AWS-сервисами.  
*Цена:* $19/мес.

### **Droid**
*Разработчик:* Factory | *Лицензия:* Проприетарная  
Промышленный робот для автоматизации разработки. Сам разбирает задачи из Jira, пишет код, создаёт PR, запускает CI/CD. Рассчитан на крупные команды.  
*Цена:* N/A (корпоративные продажи).

### **Qwen Code**
*Разработчик:* Alibaba | *Лицензия:* Open source  
Агент на базе модели Qwen (от Alibaba Cloud). Работает через CLI, поддерживает несколько языков программирования. Есть бесплатный лимит через облачный API.  
*Цена:* бесплатно (лимитировано) / BYOK.

### **Devin**
*Разработчик:* Cognition | *Лицензия:* Проприетарная  
Полностью автономный «AI-инженер»: получает задачу, сам планирует, пишет код, прогоняет тесты и открывает PR, работая в облачной среде с собственным браузером и терминалом. После выхода Devin 2.0 цена снижена с $500 до $20/мес, что открыло продукт для широкой аудитории. Компания также владеет Windsurf.  
*Цена:* от $20/мес.

### **Jules**
*Разработчик:* Google | *Лицензия:* Проприетарная  
Асинхронный агент от Google, тесно интегрированный с GitHub. В отличие от реактивных агентов, **проактивен**: сам сканирует репозиторий на предмет `#TODO`, устаревших зависимостей и потенциальных улучшений и предлагает работу без явного запроса. Удобен для фоновой рутины и поддержки кодовой базы.  
*Цена:* бесплатный лимит / платные планы.

### **OpenHands**
*Разработчик:* All Hands AI (ранее проект OpenDevin) | *Лицензия:* MIT  
Ведущая **открытая** альтернатива Devin. Агент работает в песочнице, может писать код, запускать команды, пользоваться браузером. Один из лучших среди open-source по бенчмаркам (~72% SWE-bench Verified). Доступен локально (CLI/Web) и как платный управляемый Cloud.  
*Цена:* бесплатно (BYOK) / облачный тариф.

### **Augment Code**
*Разработчик:* Augment | *Лицензия:* Проприетарная  
Заточен под **большие и долгоживущие кодовые базы**. Главная фишка — проприетарный контекст-движок (~200K токенов), который индексирует весь репозиторий и даёт агенту глубокое понимание проекта. Работает как расширение для IDE и через CLI.  
*Цена:* от $30/мес.

## 🧠 Агенты с самообучением и автоисследованием

### **Hermes Agent**
*Разработчик:* Nous Research | *Лицензия:* Open source  
Самообучающийся агент, который помнит прошлые сессии и улучшает свои навыки. Может работать через CLI, веб-интерфейс, десктоп, а также Telegram/Slack/Discord. Поддерживает 300+ моделей. Умеет создавать и дообучать свои «навыки» (skills).  
*Цена:* бесплатно (BYOK).

### **Pi Agent**
*Разработчик:* Mario & Community | *Лицензия:* Open source  
Минималистичный каркас для создания собственных кодинг-агентов. Не навязывает логику — вы сами определяете цикл работы. Поддерживает 20+ LLM-провайдеров, имеет веб-контроллер для управления несколькими агентами. Расширяется через npm-пакеты.  
*Цена:* бесплатно (BYOK).

### **Pie**
*Разработчик:* Community | *Лицензия:* Open source  
Полностью совместимая с Pi, но переписанная на Rust для максимальной производительности. Мгновенный отклик, низкое потребление памяти. Может использовать локальные модели и запускать shell-команды в песочнице.  
*Цена:* бесплатно (BYOK).

### **AutoResearch (Karpathy)**
*Разработчик:* Andrej Karpathy | *Лицензия:* Open source  
«Спящий» агент для автоматических ML-экспериментов. Запускается на ночь: меняет код, обучает модель 5 минут, проверяет метрику и коммитит улучшения. Способен за сутки провести сотни итераций без участия человека. Набрал 66k+ звёзд на GitHub за первый месяц.  
*Цена:* бесплатно (BYOK).

### **Quest 1.0**
*Разработчик:* Qoder | *Лицензия:* Проприетарная  
Первый коммерческий самообучающийся агент (анонс — январь 2026). Работает по принципу **Spec-Driven Development** — сначала пишет формальную спецификацию (границы требований, выбор технологий, критерии приёмки), затем код под неё, а при сбоях входит в цикл «reflect-repair-verify». В мае 2026 вышла версия 1.0: продукт превратился из AI-IDE в **Autonomous Development Desktop** с маршрутизацией подзадач между разными SOTA-моделями.  
*Цена:* N/A (коммерческий продукт).

### **Forge**
*Разработчик:* Сообщество | *Лицензия:* Open source  
Промышленная автономная система для «забудь и работай». Имеет трёхфазный цикл: планирование → сборка → обучение и интеграция. Каждый агент работает в изолированной облачной песочнице (E2B). Поддерживает 20+ LLM-провайдеров.  
*Цена:* бесплатно (BYOK).

### **Devika**
*Разработчик:* Сообщество | *Лицензия:* Open source  
Агент-программист, который понимает высокоуровневые инструкции («сделай веб-приложение с авторизацией») и сам разбивает на задачи, планирует сроки, пишет код, тестирует и исправляет ошибки. Поддерживает 15+ языков программирования.  
*Цена:* бесплатно (BYOK).  
> [!warning] Статус
> Devika была популярна в 2024 как ранний open-source клон Devin, но развитие фактически остановилось. Сейчас в этой нише живой и активной альтернативой является **OpenHands**.

## 💎 Какой агент выбрать?

| Если вам нужно… | Рекомендация |
| :--- | :--- |
| Простое автодополнение | GitHub Copilot, Cursor |
| Автономный агент в терминале | Claude Code, Codex CLI, Aider, OpenCode |
| Полностью автономный «инженер» (задача → PR) | Devin, OpenHands, Droid |
| Работа с очень большой кодовой базой | Augment Code, Claude Code |
| Проактивная поддержка репозитория (TODO, апдейты) | Jules |
| Самообучение и память между сессиями | Hermes Agent |
| Полный контроль над каждым шагом | Pi / Pie |
| Проведение ML-экспериментов без человека | AutoResearch |
| Промышленная автоматизация задач | Forge, Droid, Quest 1.0 |
| Открытая альтернатива Devin (self-hosted) | OpenHands |
| Бесплатный агент с открытым кодом и BYOK | Любой из Open Source в таблице |

> [!tip] Практика 2026 года
> Опытные разработчики редко выбирают **один** агент — чаще «стек» из 2–3: например, Cursor/Copilot для повседневной работы в IDE, Claude Code или Codex CLI во втором терминале для тяжёлого рефакторинга, и Devin / Codex Cloud для фоновых задач.
