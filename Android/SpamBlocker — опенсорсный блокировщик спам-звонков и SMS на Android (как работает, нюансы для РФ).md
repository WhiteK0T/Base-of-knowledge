# 📵 SpamBlocker — опенсорсный блокировщик спам-звонков и SMS на Android

[**SpamBlocker**](https://github.com/aj3423/SpamBlocker) (*aj3423*, **Kotlin**, лицензия **MIT**, ~1.7k★, живой — v5.13 от 08.07.2026) — открытый блокировщик нежелательных звонков и SMS для **Android 10+**. Главный плюс перед «звонилками» вроде Truecaller: **не требует root**, **не нужно делать его приложением по умолчанию** для звонков/SMS, а офлайн-режим **ничего не отправляет наружу**. Работает через штатный **CallScreeningService** Android.

## ⚙️ Как это работает

- **Звонки** — SpamBlocker выступает как Caller-ID приложение через **`CallScreeningService`** (официальный API Android). Не нужно быть «дефолтной звонилкой», не нужен root, после настройки может **не висеть в фоне**.
- **SMS** — два режима:
  - **Standalone** — перехватывает уведомления SMS; работает с любым SMS-приложением, но нужно разрешение на чтение SMS;
  - **Screening Provider** — как сервис для SMS-приложений, поддерживающих «SMS screening protocol»; **без разрешения на SMS**, но требует поддержки со стороны приложения.

## ✨ По каким признакам блокирует

Решение «пропустить/заблокировать» собирается из множества правил:

- **Контакты**: разрешить существующим контактам, группам контактов, префиксам.
- **Ручные правила (regex)**: блокировать по маске номера — например, всё, что начинается на `8800`/`8495`, или SMS со словами «кредит», «акция», подозрительные ссылки.
- **Эвристики**: повторные звонки с одного номера, потоки OTP-кодов подряд, звонки после набора экстренного номера, во время видеозвонка, от недавно использованных приложений (**курьер из доставки дозвонится** — как и обещает пост).
- **Attestation (STIR/SHAKEN)**: блок при провале верификации подписи вызова.
- **Базы спам-номеров**: подключаемые публичные базы (например, [DNC](https://www.ftc.gov/policy-notices/open-government/data-sets/do-not-call-data)).
- **Онлайн-запрос в реальном времени**: проверка входящего номера по нескольким API сразу (например, [PhoneBlock](https://phoneblock.net/)); можно и **репортить** спам, пополняя краудсорс-базу.
- **Прочее**: геолокация/оператор, календарь, расписание, отображаемое имя (CNAP).

## 🧪 Важный нюанс для России

> [!note] Онлайн-базы — западные, ставка на ручные правила
> Встроенные источники заточены под США/ЕС: **DNC** — реестр FTC (США), **PhoneBlock** — немецкий. **STIR/SHAKEN** в РФ операторами не применяется, так что этот признак у нас почти не сработает. Для российских номеров реально работают: **ручные regex-правила** (`8800`, `8495`, короткие номера рассылок), **эвристики** (повторные дозвоны, OTP-флуд), **белый список контактов** и **keyword-фильтр SMS** («кредит», «акция», «выиграли», ссылки). Именно на это и напирает пост — и это правильный акцент: «умная авто-магия» здесь во многом = твои же правила.

## 🔒 Приватность

> [!info] Офлайн — тишина; онлайн — уходит только номер
> - **Офлайн-функции** (свои правила, контакты, эвристики) — **не собирают и не отправляют никаких данных**.
> - **Онлайн-функции** (скачивание баз, instant-query, репорт) — API-эндпоинт видит твой **IP**, TLS/TCP-фингерпринт (выдаёт версию Android) и **проверяемый номер + код страны**. Онлайн можно **полностью отключить** (убрать разрешение INTERNET).
> - Есть **воспроизводимая сборка** (reproducible APK) — можно убедиться, что APK собран из этих исходников.

## 📦 Установка

- **[F-Droid](https://f-droid.org/packages/spam.blocker)** (рекомендуется — автообновления, проверенная сборка) · **[Obtainium](https://github.com/ImranR98/Obtainium)** (обновления прямо из GitHub-релизов) · **APK** из [GitHub Releases](https://github.com/aj3423/SpamBlocker/releases/latest).
- **Не из Google Play**: при установке Play показывает предупреждение безопасности (звонилки/SMS-фильтры под подозрением у Google) — это известная «проблема», не признак вредоносности.
- Требуется **Android 10+**.

## 🖥️ По системам

Это **приложение под Android** — к десктопным системам владельца (Gentoo/Debian/Arch) отношения не имеет, отдельного клиента под них нет. На роутере **RT-AX56U** сотовые звонки/SMS не заблокировать в принципе (это уровень оператора/телефона); смежная задача сетевого уровня — резать рекламу/трекеры по DNS, для чего есть [Pi-hole](../Apps/Pi-hole%20%E2%80%94%20%D1%81%D0%B5%D1%82%D0%B5%D0%B2%D0%BE%D0%B9%20%D0%B1%D0%BB%D0%BE%D0%BA%D0%B8%D1%80%D0%BE%D0%B2%D1%89%D0%B8%D0%BA%20%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D1%8B%20%D0%B8%20%D1%82%D0%B5%D0%BB%D0%B5%D0%BC%D0%B5%D1%82%D1%80%D0%B8%D0%B8%20%28DNS-sinkhole%29.md)/AdGuard Home.
