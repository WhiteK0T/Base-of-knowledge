# 📱 web-to-app — сборка Android-APK из сайта прямо на телефоне («карманная APK-мастерская»)

[**web-to-app** (WebToApp)](https://github.com/shiaho777/web-to-app) — нативное **Android-приложение**, которое заворачивает любой сайт (URL, локальный HTML, медиапроект) в **отдельный APK** и **собирает + подписывает его прямо на устройстве**, без облака и ПК. Под капотом — настраиваемый WebView, локальные рантаймы, подписание APK и модули-расширения. Kotlin + Python (+ C/JNI), лицензия **Unlicense** (public domain), ~4.4k★, версия **2.1.0** (19 июня 2026).

> [!info] Это сильно больше, чем «сайт → иконка»
> Telegram-пост описывает его как простой «конвертер URL в приложение». На деле репозиторий называет себя **«карманной APK-мастерской»**: патчит шаблонный APK на бинарном уровне (AXML/ARSC), вшивает конфиг и ресурсы, обрезает лишние разрешения и **подписывает локально** (`apksig` 8.3.0) — без удалённой сборочной очереди. По умолчанию рендер — **системный WebView**; **GeckoView** (движок Firefox) — опционально, докачивается при первом использовании (в TG он подан как обязательная технология — это неточность).

## ✨ Что умеет (основное)

- **Браузер/сеть:** desktop-режим, свой User-Agent, инъекции JS/CSS, статические/PAC-прокси, **DNS-over-HTTPS**, PWA-офлайн-кеш, кастомные страницы ошибок, политика попапов.
- **Расширения/автоматизация:** встроенные модули (скачивание видео, тёмная тема, блокировка рекламы, поиск по странице), **юзерскрипты Greasemonkey/Tampermonkey**, рантайм **MV3 Chrome-расширений**, генерация модулей через AI, шаринг через QR/код (ZXing).
- **Локальные рантаймы на телефоне:** **Node.js, PHP 8.4, Python, Go, WordPress** — каждый поднимается как локальный HTTP-сервер, без выгрузки в облако.
- **Опыт приложения:** сплеш-экраны (картинка/видео), плавающее окно, кастом статус-бара, web-уведомления, deep links, оверлей-перевод (20 языков).
- **Сборка/подпись APK:** свои package name/версия/иконка/архитектура, экспорт AAB с переписыванием targetSdk под Play, выбор схем подписи **V1/V2/V3**, импорт хранилищ ключей PKCS12/PFX/JKS/BKS.

> [!warning] targetSdk = 28 и почему host-приложения нет в Google Play
> Само приложение намеренно держит **targetSdk = 28**, чтобы иметь право форкать нативные бинарники (Node/PHP/Python/Go) из своего хранилища — современный Android/Play это запрещает. Поэтому **host-приложение раздаётся только через GitHub Releases**, не через Play Store. Сгенерированные web/html/медиа-приложения переписывают targetSdk до 35+ и пригодны для Play; но приложения, форкающие нативные рантаймы, остаются **APK-only**.

> [!caution] Часть функций — двойного назначения
> Среди «продвинутых» возможностей есть **анти-отладка/анти-Frida**, маскировка отпечатка браузера (28 векторов), «device disguise», «Icon Storm», «BlackTech», клонирование/маскировка ярлыков. Авторы сами помечают их как **«для технической демонстрации, с осознанного согласия пользователя»**. Маскировка приложений и подмена отпечатков легко уходит в обман пользователя/обход защит — используй только на своих устройствах и не для введения людей в заблуждение. Подписанный «самосбор» APK = бери на себя ответственность за его содержимое.

## 📦 Установка / сборка

| Что | Как |
| :--- | :--- |
| **Готовый APK** (обычный путь) | скачать из [GitHub Releases](https://github.com/shiaho777/web-to-app/releases) и поставить на Android (Play Store его не раздаёт, см. выше) |
| **Сборка из исходников** | `git clone … && cd web-to-app && ./gradlew assembleDebug` — нужен Android Studio Hedgehog+, **JDK 17**, Gradle 9.4.1 (через wrapper) |

| Система владельца | Применимость |
| :--- | :--- |
| **Android-телефон** | целевая платформа — ставится APK из Releases |
| **Gentoo / Debian-Ubuntu / Arch** (десктоп) | напрямую не нужно; пригодится только если **собираешь APK сам** — поставить Android Studio/SDK + JDK 17 (на Gentoo `dev-java/openjdk:17`, на Debian/Ubuntu `apt install openjdk-17-jdk`, на Arch `pacman -S jdk17-openjdk` + Android SDK из AUR) |
| **Entware / RT-AX56U** | **N/A** — Android-приложение, к роутеру отношения не имеет |

## 💡 Кому полезно

- Быстро **завернуть свой сайт/PWA/локальный веб-проект в APK** без сборочного ПК — прямо с телефона.
- Поиграться с **локальными рантаймами** (Node/PHP/Python/Go/WordPress) и юзерскриптами/расширениями на Android.
- **Не** для маскировки приложений под чужие и обмана пользователей — за это отвечаешь ты.
