# 📡 ESP32 — обзор и выбор варианта (2026)

> [!info] Об этой заметке
> Это **конспект своими словами** по мотивам гайда [«ESP32 Selection Guide — 2026» (DroneBot Workshop)](https://dronebotworkshop.com/esp32-2026/). Текст и факты переработаны и дополнены из открытых источников; авторские права на оригинальную статью и её изображения принадлежат DroneBot Workshop — здесь даётся ссылка на оригинал, а не его перевод.

## Что такое ESP32

**ESP32** — семейство дешёвых микроконтроллеров (SoC) от китайской **Espressif Systems** со встроенной беспроводной связью (Wi-Fi и/или Bluetooth/802.15.4). Развитие популярного ESP8266; де-факто стандарт для DIY-электроники и IoT: огромная экосистема, поддержка в Arduino IDE и ESP-IDF, цена от ~$2–10 за плату.

Главная мысль гайда 2026 года: **«ESP32» — это уже не один чип, а целое семейство из 11 вариантов** (System-on-Chip), и правильный выбор зависит от задачи. Старые двухъядерные Xtensa соседствуют с новыми одно/двухъядерными **RISC-V**, появилась поддержка **Wi-Fi 6, Thread/Zigbee и Matter**.

## Два процессорных «лагеря»

- **Xtensa** (LX6/LX7) — классические ядра: оригинальный ESP32, S2, S3. Максимум производительности и зрелая экосистема.
- **RISC-V** — новые экономичные и специализированные чипы: C2, C3, C5, C6, C61, H2, H4, P4. Дешевле, энергоэффективнее, с современной беспроводной частью.

## Сравнение всех 11 вариантов

| Чип | Ядро / архитектура | Такт | SRAM | Wi-Fi | Bluetooth | Zigbee/Thread | Лучше всего для / примечание |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **ESP32** (ориг.) | 2× Xtensa LX6 | 240 МГц | 520 КБ | 2.4 ГГц (Wi-Fi 4) | **Classic** + BLE 4.2 | — | Единственный с BT Classic; зрелая экосистема. Для новых проектов не рекомендуется |
| **ESP32-S2** | 1× Xtensa LX7 | 240 МГц | 320 КБ | 2.4 ГГц (Wi-Fi 4) | нет | — | Есть **DAC**, USB-OTG; **нет Bluetooth и MCPWM** |
| **ESP32-S3** | 2× Xtensa LX7 | 240 МГц | 512 КБ | 2.4 ГГц (Wi-Fi 4) | BLE 5.0 | — | **«Рабочая лошадка»**, AI-ускоритель (PIE/TinyML), камера+дисплей, USB-OTG |
| **ESP32-C2** | 1× RISC-V | 120 МГц | 272 КБ | 2.4 ГГц (Wi-Fi 4) | BLE 5.0 | — | Самый дешёвый, замена ESP8266; всего 14 GPIO, без USB |
| **ESP32-C3** | 1× RISC-V | 160 МГц | 400 КБ | 2.4 ГГц (Wi-Fi 4) | BLE 5.0 | — | Хороший бюджетный Wi-Fi+BLE; популярен в ESPHome |
| **ESP32-C5** | 1× RISC-V | 240 МГц | 400 КБ | **2.4/5 ГГц (Wi-Fi 6)** | BLE 5.0 | **Да (Matter)** | **Единственный с 5 ГГц Wi-Fi** (новинка) |
| **ESP32-C6** | 1× RISC-V | 160 МГц | 512 КБ | 2.4 ГГц (**Wi-Fi 6**) | BLE 5.3 | **Да (Matter)** | **Лучший универсал для IoT/умного дома** |
| **ESP32-C61** | 1× RISC-V | 160 МГц | 256 КБ | 2.4 ГГц (**Wi-Fi 6**) | BLE 5.0 | — | Бюджетная версия C6 **без Zigbee/Thread**; как co-processor |
| **ESP32-H2** | 1× RISC-V | 96 МГц | 320 КБ | **нет** | BLE 5.0 | **Да (Matter)** | Выделенный endpoint Zigbee/Thread; сверхнизкое потребление |
| **ESP32-H4** | 2× RISC-V | 96 МГц | 320 КБ | **нет** | **BLE 5.4** (+ LE Audio) | **Да (Matter)** | Новейший H-серии, лучший Bluetooth; без Wi-Fi |
| **ESP32-P4** | 2× RISC-V (+LP-ядро) | 400 МГц | 768 КБ | — (нужен компаньон) | нет | — | **Мультимедиа-флагман**: камера/дисплей MIPI, H.264, USB 2.0 |

> ТТХ — ориентир (зависят от ревизии и объёма flash/PSRAM конкретной платы). C5 и H4 на момент выхода гайда — свежие новинки с пока ограниченной программной поддержкой и выбором плат.
>
> Espressif выкладывает [zip с детальной таблицей сравнения (Feature Matrix 2026)](https://s3.us-east-1.amazonaws.com/download.dronebotworkshop.com/ESP32-Feature-Matrix-2026+-+DroneBot+Workshop.zip) — там ещё столбцы по GPIO, ADC/DAC, USB-OTG, интерфейсам камеры/дисплея и совместимости с ESPHome.

## Как выбрать (короткая логика гайда)

- **Не знаешь — бери S3.** Двухъядерный, USB-OTG, больше всего GPIO, AI-ускорение — закроет большинство проектов.
- **Умный дом / Matter** → **C6** (Wi-Fi 6 + Zigbee + Thread в одном чипе), либо **C5**, если нужен 5 ГГц.
- **Только Zigbee/Thread, без Wi-Fi** (датчики, mesh, endpoint) → **H2**; нужен передовой BLE 5.4 / LE Audio → **H4**.
- **Максимально дёшево и просто** (замена ESP8266) → **C3** (или **C2** — самый дешёвый), либо **C61** для дешёвого Wi-Fi 6.
- **Нужен BT Classic** (старые устройства) → только оригинальный **ESP32**.
- **Нужен DAC, без Bluetooth** → **S2**.
- **Экран/видео, мультимедиа** → **P4** (радио добавляется отдельным чипом-компаньоном, обычно C6).

## Платы по категориям

Гайд делит рекомендованные платы на **5 категорий** (для каждой — свои чипы):

| Категория | Рекомендуемые чипы | Примеры плат |
| :--- | :--- | :--- |
| **Универсал / эксперименты** | S3, C3, C5 | Arduino Nano ESP32, Seeed XIAO ESP32-S3, Espressif **S3-DevKitC-1**, SparkFun Thing Plus S3, Adafruit S3 Feather |
| **Дисплей и мультимедиа** | S3, P4 | Waveshare S3 Touch LCD 2″, Adafruit S3 Reverse TFT Feather, LilyGo **T-Display S3**, Waveshare **P4-NANO**, Espressif P4 Function-EV |
| **Компактные / портативные** | S3, C3, C6 | SparkFun C6 Qwiic Pocket, Seeed **XIAO** (C6/S3), Adafruit **QT Py** (C3/S3) |
| **IoT / умный дом** | C6, H2, C5, H4 | Seeed XIAO C5, DFRobot FireBeetle 2 C5, Espressif **H2-DevKitM-1**, SparkFun Thing Plus C6, Seeed XIAO C6 |
| **Бюджет / старт** | ESP32 (ориг.), C3, S3, C6 | Seeed XIAO C3, Espressif S3-DevKitC-1, Espressif C6-DevKitC-1, Adafruit QT Py S3, DFRobot FireBeetle 2 ESP32-E |

Форм-факторы, которые часто встречаются: **DevKitC** (полноразмерная отладка, максимум выведенных GPIO), **XIAO** (21×17.5 мм, очень компактно), **Feather / Thing Plus** (совместимость с шилдами, разъём LiPo), **FireBeetle** (много GPIO + зарядка LiPo). Разъёмы **Qwiic / STEMMA QT** — быстрое подключение I²C-периферии. Многие IoT-платы совместимы с **ESPHome** (no-code для Home Assistant).

> Под P4 (и частично H2/H4) Arduino IDE пока поддерживается ограниченно — для серьёзной работы используется **ESP-IDF**.

## С чего начать

**Платы.** Удобнее всего готовые DevKit-платы (`ESP32-DevKitC`, `ESP32-S3-DevKitC`, `*-C6-DevKitC` и т.п.) — на них уже есть USB-UART, регулятор питания и кнопки.

**Arduino IDE.** Добавить URL менеджера плат Espressif в *File → Preferences → Additional Boards Manager URLs*:

```
https://espressif.github.io/arduino-esp32/package_esp32_index.json
```

Затем *Tools → Board → Boards Manager* → установить **esp32 by Espressif Systems**, выбрать свою плату и порт.

**ESP-IDF.** Официальный фреймворк Espressif (C/C++, FreeRTOS) — для полного контроля и продвинутых фич; тяжелее в освоении, чем Arduino.

**Мигающий светодиод (Arduino):**

```cpp
#define LED_PIN 2   // встроенный светодиод на многих DevKit (уточни для своей платы)

void setup() {
  pinMode(LED_PIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_PIN, HIGH);
  delay(500);
  digitalWrite(LED_PIN, LOW);
  delay(500);
}
```

> [!tip] Пины различаются
> Номер встроенного LED и доступные GPIO зависят от варианта и платы (например, у многих S3/C6 это RGB-светодиод на отдельном пине). Сверяйся с распиновкой конкретной платы.
